Examples of Spacer Snippets

utility_spacers_vertical

{{{ snippet "utility_spacers_vertical" height }}}

A simple vertical spacer that adds height between elements.

Variables:

  1. height: Height in pixels

Usage:

{{{ snippet "utility_spacers_vertical" '10' }}}

utility_spacers_dividers

{{{ snippet "utility_spacers_dividers" width borderTop borderBottom paddingTop paddingBottom }}}

A horizontal divider with configurable borders and padding for visual separation between content sections.

Variables:

  1. width: Width in pixels
  2. borderTop: Top border style (e.g., "3px solid #000000")
  3. borderBottom: Bottom border style (e.g., "3px solid #000000")
  4. paddingTop: Top padding in pixels
  5. paddingBottom: Bottom padding in pixels

Usage:

{{{ snippet "utility_spacers_dividers" '540' '3px solid #000000' '3px solid #000000' '5' '7' }}}

Unified Spacing Component

utility_spacing

{{{ snippet "utility_spacing" height width borderTop borderBottom paddingTop paddingBottom }}}

A flexible spacing component that can create both vertical spaces and dividers with configurable borders.

Variables:

  1. height: Height in pixels (optional)
  2. width: Width in pixels (optional, defaults to 100%)
  3. borderTop: Top border style (optional, e.g., "3px solid #000000")
  4. borderBottom: Bottom border style (optional, e.g., "3px solid #000000")
  5. paddingTop: Top padding in pixels (optional)
  6. paddingBottom: Bottom padding in pixels (optional)

Usage Examples:

Simple vertical spacer:

{{{ snippet "utility_spacing" '20' }}}

Double-bordered divider:

{{{ snippet "utility_spacing" '' '540' '3px solid #000000' '3px solid #000000' '10' '10' }}}

Single top border with spacing:

{{{ snippet "utility_spacing" '30' '' '1px solid #CCCCCC' '' '15' }}}

Custom width divider:

{{{ snippet "utility_spacing" '' '200' '2px dashed #999999' '' '5' '5' }}}

Large spacing with bottom border:

{{{ snippet "utility_spacing" '50' '' '' '1px solid #EEEEEE' '' '20' }}}

Notes:

  • All parameters are optional
  • Width defaults to 100% if not specified
  • Can be used for both spacing and dividers
  • Email-client safe with table-based structure
