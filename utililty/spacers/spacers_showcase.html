Examples of Spacer Snippets

utility_spacers_vertical

{{{ snippet "utility_spacers_vertical" height }}}

A simple vertical spacer that adds height between elements.

Variables:

  1. height: Height in pixels

Usage:

{{{ snippet "utility_spacers_vertical" '10' }}}

utility_spacers_dividers

{{{ snippet "utility_spacers_dividers" width borderTop borderBottom paddingTop paddingBottom }}}

A horizontal divider with configurable borders and padding for visual separation between content sections.

Variables:

  1. width: Width in pixels
  2. borderTop: Top border style (e.g., "3px solid #000000")
  3. borderBottom: Bottom border style (e.g., "3px solid #000000")
  4. paddingTop: Top padding in pixels
  5. paddingBottom: Bottom padding in pixels

Usage:

{{{ snippet "utility_spacers_dividers" '540' '3px solid #000000' '3px solid #000000' '5' '7' }}}
