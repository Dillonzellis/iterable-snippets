Examples of Typography Snippets

typography_text

{{{ snippet "typography_text" alignment fontSize lineHeight letterSpacing fontWeight fontFamily textTransform textColor paddingTop paddingRight paddingBottom paddingLeft isLink linkUrl isUnderlined text }}}

A comprehensive typography snippet for all text styling needs including optional link wrapping.

Variables:

  1. alignment: Text alignment (left, center, right)
  2. fontSize: Font size in pixels
  3. lineHeight: Line height in pixels
  4. letterSpacing: Letter spacing in em
  5. fontWeight: Weight of the font
  6. fontFamily: Font family (include fallbacks)
  7. textTransform: Text transformation (uppercase, lowercase, capitalize, none)
  8. textColor: Hex color code for text
  9. paddingTop: Padding top in pixels
  10. paddingRight: Padding right in pixels
  11. paddingBottom: Padding bottom in pixels
  12. paddingLeft: Padding left in pixels
  13. isLink: Boolean for if text should be wrapped in link
  14. linkUrl: URL if isLink is true
  15. isUnderlined: Boolean for if link should be underlined
  16. text: The text content to display

Usage:

{{{ snippet "typography_text" 'center' '45' '50' '0' '900' 'Arial' 'uppercase' '#004fff' '0' '0' '0' '0' '' '' 'false' 'Sample Text' }}}
